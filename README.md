# Disco Web

Made using MongoDB, Express, React & Node.

# Get Started - Dev Env
Go into project folder: 
```
    ~            cd server
    ~/server:    npm install
    ~/server:    PORT=3001 node bin/www
    ~/server:    cd ../
    ~            cd client
    ~            npm install
    ~            npm start
```
Front-end: [localhost:3000](localhost:3000)

API: [localhost:3001](localhost:3001)
